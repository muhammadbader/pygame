# Audio Assets

This directory contains sound effects for the Desert Serpent game.

## Required Audio Files

The following audio files are needed for the full game experience:

1. **eat.mp3** - Sound effect when the snake eats food (treasure)
   - Suggested: A pleasant chime or coin collection sound
   - Duration: ~0.5 seconds

2. **game_over.mp3** - Sound effect when the game ends
   - Suggested: A dramatic mystical sound or game over tone
   - Duration: ~1-2 seconds

3. **move.mp3** - Sound effect for snake movement (optional)
   - Suggested: A subtle slither or whoosh sound
   - Duration: ~0.2 seconds

## Note

The game will work without these audio files. Sound effects are loaded gracefully and the game will continue to function normally if the files are not present.

To add sounds:
1. Find or create MP3 audio files matching the names above
2. Place them in this directory
3. The game will automatically load and play them
